Безопасная и стабильная RSS-лента для Яндекс.Дзена с учётом требований модерации и особенностей WordPress (Gutenberg).

== Description ==

X-Tiger RSS for Dzen — плагин для WordPress, который создаёт отдельную RSS-ленту, корректно подготовленную для публикации материалов в Яндекс.Дзене.

Плагин ориентирован на редакционные и аналитические материалы и учитывает типовые причины отказов модерации Дзена: некорректную разметку, коммерческий характер контента, проблемы с изображениями и очисткой HTML.

Основные возможности:
* отдельный RSS endpoint `/dzen/rss.xml`
* выбор категории WordPress для публикации в RSS
* настраиваемые `<title>`, `<link>` и `<description>` RSS-канала
* поддержка `content:encoded`
* корректная очистка HTML через `wp_kses` (совместимо с Gutenberg)
* автоматическая проверка наличия и размера изображения (≥700px)
* минимальная длина текста статьи
* выбор категории контента Дзена (`business`, `marketing`, `technology` и др.)
* превью RSS в админке
* стабильная работа без «пустой ленты»

Плагин не добавляет рекламу, не модифицирует контент и не внедряет сторонние скрипты. Он только формирует корректный RSS.

== Installation ==

1. Загрузите ZIP-архив плагина через админку WordPress  
2. Активируйте плагин  
3. Перейдите в «Настройки → X-Tiger RSS for Dzen»  
4. Укажите:
   * рубрику WordPress для Дзена
   * заголовок и ссылку RSS-канала
   * описание канала
   * лимит статей
5. Используйте адрес `/dzen/rss.xml` при добавлении RSS в Яндекс.Дзен

== Usage ==

1. Публикуйте статьи в выбранную рубрику (например, `dzen`)
2. Убедитесь, что у статьи:
   * есть изображение шириной не менее 700px
   * текст носит информационный или аналитический характер
3. Проверьте превью RSS в настройках плагина
4. Отправьте RSS-ленту на модерацию Дзена

Плагин всегда показывает последние N статей и не удаляет их из RSS.
