# x-tiger-dzen-rss
Безопасная RSS-лента для Дзена с жёсткой валидацией контента и защитой от модерационных отказов. Плагин создаёт отдельную RSS-ленту для Дзена по всем актуальным требованиям платформы. Решение предназначено для сайтов, которые хотят автоматизировать публикацию контента в Дзене без отказов модерации.
