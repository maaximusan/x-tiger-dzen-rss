Безопасная RSS-лента для Дзена с жёсткой валидацией контента и защитой от модерационных отказов.

== Description ==

Плагин создаёт отдельную RSS-ленту для Дзена по всем актуальным требованиям платформы.

Возможности:
* отдельный endpoint `/dzen/rss.xml`
* публикация только из категории `dzen`
* строгая очистка HTML
* проверка изображений (≥700px)
* защита от повторной отправки
* режимы `native` и `native-draft`
* админка с логом отправленных материалов

Решение предназначено для сайтов, которые хотят автоматизировать публикацию контента в Дзене без отказов модерации.

== Installation ==

1. Загрузите ZIP-архив плагина через админку WordPress
2. Активируйте плагин
3. Перейдите в «Настройки → RSS для Дзена»
4. Скопируйте RSS-адрес `/dzen/rss.xml` в Студию Дзена

== Usage ==

Добавляйте статьи в категорию `dzen`.
Плагин автоматически подготовит корректную RSS-ленту.

Рекомендуемый режим — `native-draft` для первичной модерации.
